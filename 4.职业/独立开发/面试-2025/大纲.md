# 大纲

得从平台看看大厂都需要哪些技术能力

## 基础

1. java
2. kotlin
3. 数据库
4. js
5. leetcode 算法

## 专业深度

1. gradle构建
2. 三方框架：rxjava、Retrofit、Glide
3. 插件化、换肤
4. AOP、注解处理等动态编译、工具建立
5. framework核心原理[重点]
6. ndk开发
7. aosp 模块编译
8. 性能优化[重点]：
    1. 异常、卡顿统计
    2. 问题分析：日志、工具、界面
    3. 帧率
    4. 解决性能问题的多种姿势
9. 单元测试

## 架构

一切都为了保障交付质量。

1. 团队测试驱动开发模式（TDD）建设
2. 质量保障
3. 性能分析报告
4. 线上问题复盘，责任划分
5. 需求分析、技术方案
6. 对于不可接受需求怎么向上升级，保证需求推进无问题。
7. 编码规范统一

## 管理能力

1. 项目进度、开发进度、开发排期等等
2. 沟通能力
3. 团队技术演进、人员培养
