>这个教程很老，仅做参考

如果只要求快速开发应用，掌握以下内容即可：

1. 基础语言：[Objective-C](./Objective-C)、[Swift](./Swift)
2. UI库：UIKit（最基础的库，OC、Swift项目都要用）、snapkit（三方库，使用最多，封装了布局的处理）、SwiftUI（新手直接学这个就行，类似kotlin的compose）
3. 包管理器：cocoapods
