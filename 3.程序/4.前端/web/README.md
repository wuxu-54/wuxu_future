## 早知道

1. nodejs ： 基于Chrome V8 JavaScript引擎的JavaScript运行时环境，可以用js编译服务器代码，实现后端服务的开发。
2. vite、webpack ：前端程序化的打包工具。（类似android的gradle、javaweb的maven）
3. Axios：js网络框架
4. Typescript ：新语言，JS语言的超类，完全兼容JS，但又做了特性化处理。
