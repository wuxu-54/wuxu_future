# pinia处理异步操作

对于异步操作，你可以在 actions 中使用 async/await 语法来处理异步逻辑，并在操作完成后更新 state。

```js
// 在定义 Store 时
export const useMyStore = defineStore('myStore', {
  state: () => ({
    data: null
  }),
  actions: {
    async fetchData() {
      const response = await fetch('some-api-url');
      const data = await response.json();
      this.data = data;
    }
  }
});

// 在组件中使用
const myStore = useMyStore();
myStore.fetchData();
```
