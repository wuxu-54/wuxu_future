# TextButton问题

## TextButton文字居中

实际开发中，遇到一个ui问题：TextButton文字无法居中。如何解决？

答：实测以下这种方法有效：

```dart
  TextButton(
    style: ButtonStyle(
    backgroundColor:
        MaterialStateProperty.all(Colors.blue),
    // 设置背景色
    foregroundColor:
        MaterialStateProperty.all(Colors.white),
    // 设置文本颜色
    overlayColor: MaterialStateProperty.all(
        Colors.black.withOpacity(0.1)),
    // 设置按下时的背景色
    textStyle: MaterialStateProperty.all(TextStyle(
        fontSize: 18,
        fontWeight: FontWeight.bold)), // 设置文本样式
    ),
    onPressed: () {
    _loginIn(context);
    },
    child: Text(
    WxLocalizations.languageResString(context).login,
    strutStyle: StrutStyle( //这句会使文字居中！类似css 设定行高，将文本居中
        // 使用StrutStyle 设置行高，使得文本居中对齐，否则会有对不齐的问题
        fontSize: 18,
        height: 1.1,
        forceStrutHeight: true),
    ))
```
