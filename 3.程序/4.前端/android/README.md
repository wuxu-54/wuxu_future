## AOSP

### 源码阅读

思考：

1. 为什么要这么做？符合的设计模式、架构思想，这样做有哪些好处、能不能更好。
2. 我会怎么做

### Framework

- 推荐书籍：《深入理解Android卷1-3》 邓凡平著
- 学习流程：
    1. 《深入理解Android》阅读
    2. 清华镜像下载源码，阅读源码 从native层到java层
    3. 绘制流程图、结构图
    4. 试着修改、编译 so文件，条件允许试着通过AOSP编译完整系统
