# jni数值传递

## int 传递

* java层

```java
public class JNIPassData {
    static {
       System.loadLibrary("ndk1");
    }

    public native int passInt(int value);

    public native String passString(String str);

    public native int[] passArray(int[] arr);
}
```

* native层

```c
#include <jni.h>
#include <string.h>

JNIEXPORT jint JNICALL
Java_com_wuxu_ndk1_JNIPassData_passInt(
        JNIEnv* env,
        jobject thiz ,jint int_value) {

    return 10+int_value;
}

JNIEXPORT jstring JNICALL
Java_com_wuxu_ndk1_JNIPassData_passString(JNIEnv *env, jobject thiz, jstring str) {
    return "";
}

JNIEXPORT jintArray JNICALL
Java_com_wuxu_ndk1_JNIPassData_passArray(JNIEnv *env, jobject thiz, jintArray arr) {
    return arr;
}
```
