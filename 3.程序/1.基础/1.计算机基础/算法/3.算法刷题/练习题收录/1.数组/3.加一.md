# 加一(简单)

```txt
给定一个由 整数 组成的 非空 数组所表示的非负整数，在该数的基础上加一。

最高位数字存放在数组的首位， 数组中每个元素只存储单个数字。

你可以假设除了整数 0 之外，这个整数不会以零开头。

 

示例 1：

输入：digits = [1,2,3]
输出：[1,2,4]
解释：输入数组表示数字 123。
示例 2：

输入：digits = [4,3,2,1]
输出：[4,3,2,2]
解释：输入数组表示数字 4321。
示例 3：

输入：digits = [9]
输出：[1,0]
解释：输入数组表示数字 9。
加 1 得到了 9 + 1 = 10。
因此，结果应该是 [1,0]。
 

提示：

1 <= digits.length <= 100
0 <= digits[i] <= 9
```

思路一（优选），从后往前，如果等于9那么置为0，不等于9直接自增并返回，要考虑首位是9的情况，需要将数组扩容。

```java
class Solution {
    public int[] plusOne(int[] digits) {
        int addN = 1;
        for(int i = digits.length-1;i>=0;i--){
            if(i == 0 && digits[0] == 9 && addN==1){
                int[] newData =  new int[digits.length+1];
                newData[0] =1;
                return newData;
            }

            if(digits[i]!=9 && addN==1){
                digits[i] += 1;
                return digits;
            }

            if(digits[i] == 9 && addN==1){
                digits[i] = 0;
                continue;
            }

        }

        return digits;
    }
}
```

思路二：从后往前遍历，找到最后一个不等于9的数，并把后面的数全部置为0。

```java
class Solution {
    public int[] plusOne(int[] digits) {
        int n = digits.length;
        for (int i = n - 1; i >= 0; --i) {
            if (digits[i] != 9) {
                ++digits[i];
                for (int j = i + 1; j < n; ++j) {
                    digits[j] = 0;
                }
                return digits;
            }
        }

        // digits 中所有的元素均为 9
        int[] ans = new int[n + 1];
        ans[0] = 1;
        return ans;
    }
}
```
