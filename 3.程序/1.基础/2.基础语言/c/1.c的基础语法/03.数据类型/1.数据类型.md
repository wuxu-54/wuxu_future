# 数据类型

参考：[菜鸟教程](https://www.runoob.com/cprogramming/c-data-types.html)

1. 基本数据类型  
    它们是算术类型，包括整型（int）、字符型（char）、浮点型（float）和双精度浮点型（double）。
2. 枚举类型：  
    它们也是算术类型，被用来定义在程序中只能赋予其一定的离散整数值的变量。
3. void 类型：  
    类型说明符 void 表示没有值的数据类型，通常用于函数返回值。
4. 派生类型：  
    包括数组类型、指针类型和结构体类型。
    >NULL、0 都可以表示空指针
