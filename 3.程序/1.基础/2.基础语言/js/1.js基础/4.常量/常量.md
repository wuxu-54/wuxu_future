# 常量

## 什么是常量

常量是一种不可被修改的变量，使用 const 关键字修饰。

## 常量作用

* 当业务逻辑需要用到恒定的某些数据，可以用常量定义。

## 对比变量，常量的好处

* 当遇到恒定数据时，使用常量可以避免多次申请内存，其是一个唯一不可变的数据。
* 当想要数据不可被修改时，可以用常量。
* 尽量使用常量，如果知道变量可变，再改为let

## 使用

```javaScript
const G = 9.9
```  

## 补充

1. 变量不会赋值可以使用 const
2. 数组或对象因为存储位置原因，栈、堆 所以数组或对象变量在没有重新赋予新数组或对象时可以使用const，不论其内部元素怎么变化
