# 对象

对象object 是描述事物的一个数据结构体，java中的对象指的是一个引用类型数据。

## 对象声明

对象结构：{属性名:属性值}，声明：

```javascript
//声明对象体

//方式1
let obj = {name : "11",age : 12,'sex':"man"}

//方式2
let obj2 = new Object()
    obj2.name = "22"
```

## 对象属性使用

```javascript
//增加属性，对象.新属性=value
obj2.age=14
//改值
obj2.age  = 14
//查值，属性使用方式1
console.log(obj2.age)
//属性使用方式1 2
console.log(obj['sex'])
```

## 对象方法声明及调用

```javascript
let obj = {
        age : 12,
        //定义方法
        fun:function(){
            console.log("这是对象的方法")
        }
    }

//对象名.方法名
obj.fun()
```

## 对象变量

```javascript
let obj = {
    name:"111",
    age:123,
    address:"123"
}

//遍历用 for in ，这个方法不推荐遍历数组，因为其下标是string类型
for(let key in obj){
    console.log(key)
    //key是个字符串 obj.k相当于 obj.'name' 是个错误语法，所以需要用方式2获取对象属性值
    console.log(obj[key])
}

```
