# 函数分文件编写

分文件编写就是让函数在 .h中声明，在 .cpp 文件中定义。（这算是c、c++的语言风格了）

## 作用

分文件编写，可以让代码结构更清晰。

## 流程

1. 创建.h文件
2. 创建.cpp文件
3. 头文件中声明函数
4. 源文件中定义函数

## 实例

a.h 头文件中声明

```cpp
#include<iostream>
using namespace std;

void add(int i);
```

a.cpp源文件中定义

```cpp
#include<a.h>
//.h中声明
void add(int i){
    cout<<i<<endl;
}
```
