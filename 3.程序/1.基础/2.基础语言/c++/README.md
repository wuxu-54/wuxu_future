# C++

c++ 在c的基础上进行了扩展，增加了【引用、类、友元】， 提出了继承、抽象、封装、多态等概念，所以必须先学习c再来学c++。本文中有些基础的内容，如变量、常量、注释、命名规则等等与c一模一样。

## 学习建议

1. 学习c++ 必须要先学习c，掌握c基础，在c基础上了解c++相对c有哪些修改并掌握。
2. 本c++内容完全按照视频课程来学习的，尽量把视频里面的案例敲一遍。
