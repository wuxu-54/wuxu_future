# 使用web模块搭建本地简易服务器

[官网地址](https://webpy.org/)

## 步骤

1. 安装web模块：`pip3 install web.py`
2. 创建py文件

    ```python
    #!/usr/bin/python3
    # -*- coding: UTF-8 -*-
    import web

    urls = (
        '/(.*)', 'hello'
    )
    app = web.application(urls, globals())

    class hello:
        def GET(self, name):
            if not name:
                name = 'World'
            return 'Hello, ' + name + '!'

    if __name__ == "__main__":
        app.run()
    ```

3. 运行：`python3 app.py` 此时通过 `http://0.0.0.0:8080/` 可以访问到了
