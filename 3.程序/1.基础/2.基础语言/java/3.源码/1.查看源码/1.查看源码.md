# 查看源码

很多在线文档，没有真正的源代码，所以这里简单说下如何看源代码。

[我的源码仓库](https://gitee.com/wuxu54/wuxu_jdk.git)

## 本地源代码查看

1. 下载jdk
2. 进入jdk目录：`corretto-1.8.0_372/Contents/Home/src.zip`
![Alt text](image.png)
3. 拷贝**src.zip**压缩包并解压，此压缩包里就是源码
![Alt text](image-1.png)
