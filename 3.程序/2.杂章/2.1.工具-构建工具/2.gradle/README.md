# Gradle

Gradle是一个流行的基于JVM的构建工具，常用于Android和Kotlin项目，也支持其他JVM语言项目如web项目等。
>👉🏻：本文作为gradle学习章节。
>👉🏻：本文作为学习内容，有些知识并不全或细致、有些内容可能有冗余（因为我是对照AI大模型和文档整理的）。可以看下面的学习网站的内容（推荐看最下面的【Gradle 文档】）。
