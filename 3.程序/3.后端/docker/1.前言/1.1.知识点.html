第1周练习题

1. Docker的核心组件包括哪些？

答案：Docker的核心组件包括：Docker引擎（Docker Engine）、Docker镜像（Images）、Docker容器（Container）、Docker仓库（Registry）。
第2周练习题

1. 编写一个Dockerfile，构建一个包含Nginx的镜像。

答案： FROM nginx:latest COPY ./index.html /usr/share/nginx/html/ EXPOSE 80 CMD ["nginx", "-g", "daemon off;"]
第3周练习题

1. 如何创建一个自定义的Docker网络？

答案：使用命令 docker network create -d bridge my-network 其中 -d 指定网络驱动类型，bridge是最常用的网络类型。
第4周练习题

1. Docker Swarm的主要特点是什么？

答案：Docker Swarm的主要特点包括： - 集群管理集成进Docker Engine - 分散式设计 - 声明式服务模型 - 服务扩展和负载均衡 - 多主机网络 - 服务发现
