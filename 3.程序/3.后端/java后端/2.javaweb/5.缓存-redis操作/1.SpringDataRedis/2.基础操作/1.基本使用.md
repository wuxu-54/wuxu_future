# 使用Spring Data Redis

1. **注入RedisTemplate**

   在需要使用Redis的地方，通过`@Autowired`注解注入`RedisTemplate`对象。

2. **操作Redis数据**

   `RedisTemplate`提供了丰富的API来操作Redis中的各种数据类型，包括字符串、哈希、列表、集合和有序集合等。以下是一些常见的操作示例：

   - **操作字符串类型数据**：

     ```java
     ValueOperations<String, String> valueOperations = redisTemplate.opsForValue();
     valueOperations.set("name", "小明");
     String name = valueOperations.get("name");
     System.out.println(name); // 输出：小明
     ```

   - **操作哈希类型数据**：

     ```java
     HashOperations<String, String, String> hashOperations = redisTemplate.opsForHash();
     hashOperations.put("user", "name", "tom");
     hashOperations.put("user", "age", "20");
     String userName = hashOperations.get("user", "name");
     System.out.println(userName); // 输出：tom
     ```

   - **操作列表类型数据**：

     ```java
     ListOperations<String, String> listOperations = redisTemplate.opsForList();
     listOperations.leftPush("mylist", "a", "b", "c");
     List<String> mylist = listOperations.range("mylist", 0, -1);
     System.out.println(mylist); // 输出：[c, b, a]
     ```

   - **操作集合类型数据**：

     ```java
     SetOperations<String, String> setOperations = redisTemplate.opsForSet();
     setOperations.add("set1", "a", "b", "c");
     Set<String> set1 = setOperations.members("set1");
     System.out.println(set1); // 输出：[a, b, c]
     ```

   - **操作有序集合类型数据**：

     ```java
     ZSetOperations<String, String> zSetOperations = redisTemplate.opsForZSet();
     zSetOperations.add("zset1", "a", 1);
     zSetOperations.add("zset1", "b", 2);
     Set<String> zset1 = zSetOperations.range("zset1", 0, -1);
     System.out.println(zset1); // 输出：[a, b]（根据分数排序）
     ```
